\# Burp Suite Notes



\## What is Burp Suite?

Burp Suite is a web application security testing tool used to intercept,

inspect, and modify HTTP/HTTPS traffic between a browser and a web server.



It works as a proxy, sitting between the client (browser) and the server.



\## Why Burp Suite is Important

Burp Suite helps security professionals:

\- Understand how web applications communicate

\- Identify vulnerabilities such as SQL Injection and XSS

\- Manually test application behavior



\## Core Components

\- Proxy: Intercepts requests and responses

\- Repeater: Allows manual modification and resending of requests

\- Intruder: Automates customized attacks

\- Decoder: Encodes and decodes data formats



\## How Burp Fits in Web Security

Reconnaissance → Mapping → Burp Suite Testing → Exploitation → Reporting
