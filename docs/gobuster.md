\# Gobuster Notes



\## What is Gobuster?

Gobuster is a command-line tool used for brute-forcing

URIs (directories and files), DNS subdomains, and virtual hosts.



It is commonly used during the reconnaissance phase of web

application security testing.



\## Why Gobuster is Important

Gobuster helps security professionals:

\- Discover hidden directories and files

\- Identify exposed endpoints

\- Expand the attack surface of a target application



\## Common Gobuster Modes

\- dir: Directory and file brute-forcing

\- dns: Subdomain enumeration

\- vhost: Virtual host discovery



\## Where Gobuster Fits in Web Security

Reconnaissance → Gobuster Enumeration → Burp Suite Testing → Exploitation → Reporting
